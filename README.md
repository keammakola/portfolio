# Kea Mmakola - Portfolio Website

A modern, responsive portfolio website showcasing data engineering, machine learning, and software development expertise.

##  Features

- **Responsive Design** - Works on all devices
- **Modern Animations** - Floating elements, gradient effects, and smooth transitions
- **Contact Form** - Functional contact form with Netlify Forms integration
- **GitHub Stats** - Live GitHub statistics integration
- **Tech Stack Showcase** - Interactive technology cards with hover effects

##  Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Icons**: Ionicons
- **Fonts**: Google Fonts (Poppins)
- **Form Handling**: Netlify Forms
- **Deployment**: GitHub Pages

##  Sections

- **About** - Personal introduction and focus areas
- **Resume** - Education, experience, and technical skills
- **Portfolio** - Featured projects and work samples
- **Blog** - Technical articles and tutorials
- **Contact** - Professional contact form and information

##  Design Features

- Dark theme with orange accent colors
- Gradient backgrounds and animations
- Floating card effects
- Responsive grid layouts
- Modern typography
